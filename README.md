# Умные агенты для вашей работы

Современный корпоративный веб-сайт, посвящённый внутренним ИИ-агентам, предназначенным для помощи сотрудникам компании в работе с тендерами.

## Описание

Наши ИИ-агенты созданы для того, чтобы экономить ваше время, снижать риски и повышать качество работы с тендерами. Просто загрузите документ или вставьте ссылку — всё остальное сделает интеллектуальный помощник.

## Функциональность

- **Агент первичного анализа тендера**: Быстро оцените, стоит ли участвовать
- **Агент извлечения данных из ТЗ**: Превратите PDF в структурированное задание за секунды

## Технические особенности

- Чистый JavaScript (ES6+) без использования фреймворков
- Чистый CSS без препроцессоров
- Клиентская маршрутизация с использованием History API
- Адаптивный дизайн
- Многоступенчатая сборка Docker
- Оптимизированный образ на базе nginx:alpine

## Установка и запуск

### Сборка Docker-образа

```bash
docker build -t company-ai-agents-website .
```

### Запуск контейнера

```bash
docker run -p 80:80 company-ai-agents-website
```

После этого сайт будет доступен по адресу http://localhost

## Структура проекта

- `src/index.html` - основной HTML файл
- `src/styles.css` - стили приложения
- `src/app.js` - JavaScript логика и маршрутизация
- `Dockerfile` - инструкции для сборки Docker-образа
- `nginx.conf` - конфигурация веб-сервера

## Страницы приложения

- `/` - Главная страница
- `/agents/tender-analysis` - Страница агента анализа тендера
- `/agents/document-extraction` - Страница агента извлечения данных
